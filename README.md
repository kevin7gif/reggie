# reggie--瑞吉外卖

## 项目所涉及技术：
* 前端：vue+html+css
* 后端：springboot2+mybatisplus+mysql8+redis6+swagger
* 其中redis在Linux上部署，采用redis对登录时获取到的验证码和经常访问的数据进行缓存，避免当多个用户同时访问时，数据库压力过大。
* 具体业务功能可以查看接口文档
